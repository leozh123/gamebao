发布游戏

用 GitHub Issue 托管游戏列表，无需注册任何账号

GameBao 通过 GitHub Issues 公开游戏信息，无需任何 Token 配置。

1
点击下方按钮，用 GitHub 账号登录，自动跳转到 Issue 创建页面
2
填写游戏信息：名称 / 游戏链接 / 封面图片 / 标签 / 描述
3
点击 Submit new issue，游戏即刻上线，所有人都能看到！

需使用 GitHub 账号登录。如无账号，请前往 github.com/join 免费注册

游戏链接从哪里来？

你可以把游戏上传到 GitHub Pages、Netlify、Vercel 等免费托管平台，然后将生成的链接填在这里。
或者直接把游戏 HTML 文件的 URL 填入，只要能在浏览器中打开即可。
